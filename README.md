# Active-Ragdoll

Simple implimentation of Active-Ragdoll in Unity.

**Features**
- [x] Self balance: Forward
- [ ] Self balance: Backward
- [ ] Self balance: Left
- [ ] Self balance: Right
- [x] Walk towards target
- [x] Fall down
- [ ] Get up
